关于 About

天气盒子是一款极简天气预报应用：实时天气、24 小时逐时预报、未来 7 天预报、支持中国全部城市搜索与多城市管理、可选自动定位，界面背景随天气与昼夜变化。无广告、无账号、打开即用。
Weatherbox is a minimalist weather app: current conditions, 24-hour and 7-day forecasts, search for any city in China, multi-city management, optional auto-location, and an animated background that changes with the weather. No ads, no account.

常见问题 FAQ

  • 如何添加城市？ 点击右上角列表按钮 →“+”→ 搜索城市名称或从热门城市中选择。
    How to add a city? Tap the list button (top-right) → “+” → search or pick from popular cities.
  • 如何切换城市？ 在主页左右滑动即可在已添加城市间切换。
    Switch cities? Swipe left/right on the main screen.
  • 定位不工作？ 请在“设置 → 隐私与安全 → 定位服务 → 天气盒子”中允许定位；你也可以始终手动搜索添加城市。
    Location not working? Enable it in Settings → Privacy → Location Services → Weatherbox, or just add cities manually.
  • 天气数据来源？ 数据来自 Open-Meteo（open-meteo.com）。
    Data source? Weather data is provided by Open-Meteo.
